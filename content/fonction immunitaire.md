# Fonction immunitaire

# I. Introduction au système immunitaire

Le système immunitaire est le **système de défense naturel** de l’organisme. Il protège contre ce qui est étranger (non-soi), comme les bactéries, virus, toxines et cellules anormales.

## A. Soi et non-soi

- **Soi** : éléments appartenant à l’organisme (groupes sanguins ABO, CMH/HLA).
    
- **Non-soi** : agents étrangerrs reconnus par le système immunitaire (pathogènes, toxines, cellules modifiées).
    

## B. Types d’immunités

### **1. Immunité innée (non spécifique)**

- Présente dès la naissance.
    
- Rapide, immédiate.
    
- Dépend de barrières physiques/chimiques + réaction inflammatoire + phagocytose.
    
- **Pas de mémoire immunitaire.** (mais première étape de l'immunité acquise)
    

### **2. Immunité acquise (spécifique)**

- Se développe au cours de la vie.
    
- Réponse plus lente (first time) mais **hautement spécifique**.
    
- Possède une **mémoire immunitaire** durable (LB et LT mémoire).
    
- Utilise les **CPA** (cellules dendritiques, macrophages, LB) et le **CMH I/II** pour activer les LT.
    

## C. Agents pathogènes

- Parasites
    
- Champignons
    
- Bactéries
    
- Virus
    

---

# II. Les cellules immunitaires

Le système immunitaire repose sur différentes lignées cellulaires dérivées de l’hématopoïèse.

## A. Granulocytes

- **Neutrophiles** : phagocytose (premiers sur le site infectieux).
    
- **Éosinophiles** : allergies, parasites.
    
- **Basophiles** : libération d’histamine → inflammation.
    

## B. Mastocytes

- Présents dans les tissus.
    
- Libèrent des médiateurs inflammatoires (histamine).
    
- Centrales dans les allergies de type I (IgE).
    

## C. Monocytes / Macrophages

- Monocytes circulants → macrophages tissulaires.
    
- Phagocytent et digèrent les pathogènes.
    
- **CPA majeures** : présentent l’antigène aux LT via CMH II.
    

## D. Lymphocytes (acquis)

### **1. Lymphocytes B (LB)**

- Maturation dans la moelle osseuse.
    
- Produisent les **anticorps** (plasmocytes).
    
- Génèrent des **LB mémoire**.
    

### **2. Lymphocytes T (LT)**

- Maturation dans le **thymus**.
    
- Deux sous-types principaux :
    
    - **LT4 (auxiliaires)** : coordination immunitaire, activation LB, LT8, macrophages.
        
    - **LT8 (cytotoxiques)** : destruction des cellules infectées ou tumorales.
        
- Sous-types spécialisés :
    
    - **Th1** : défense intracellulaire (IFN-γ).
        
    - **Th2** : allergies / IgE (IL-4, IL-5).
        
    - **Th17** : antifongique / bactéries extracellulaires (IL-17).
        
    - **Treg** : tolérance immunitaire (IL-10, TGF-β).
        

---

# III. Formation et maturation des lymphocytes (hématopoïèse)

Cellule souche hématopoïétique multipotente →

- **Lignée myéloïde** : granulocytes, monocytes, mastocytes, mégacaryocytes, érythrocytes.
    
- **Lignée lymphoïde** : LB, LT, NK.
    

### Maturation

- **LB** : moelle osseuse → organes lymphoïdes.
    
- **LT** : moelle → thymus (éducation, sélection positive/négative) → organes lymphoïdes.
    

---

# IV. Immunité innée

## A. Défense externe

Barrières empêchant l’entrée des agents pathogènes :

- **Peau** : kératine, sébum.
    
- **Muqueuses** : mucus, enzymes, acides.
    
- **Sécrétions** : sueur (pH acide), larmes, bile, salive.
    
- **Flore commensale** : compétition avec les microbes.
    
- **Poils** : expulsion mécanique.
    

## B. Défense interne (inflammation)

### **Phases de la réaction inflammatoire**

1. **Vasodilatation / perméabilité** (TNF, IL-1, IL-6).
    
2. **Diapédèse** : migration des leucocytes vers les tissus.
    
3. **Phagocytose** : neutrophiles puis macrophages.
    
4. **Cicatrisation**.
    

⚠️ Si accumulation de débris : **formation de pus**.

---

# V. Immunité acquise

Réponse spécifique visant un antigène précis avec mémoire durable.

## A. Acteurs clés

- **Antigène** : substance reconnue comme étrangère.
    
- **Anticorps (Ac / immunoglobulines)** produits par LB.
    
- **CPA** : cellules dendritiques, macrophages, LB.
    
- **CMH I / CMH II** pour la présentation antigénique.
    

## B. Rôle des anticorps

- Neutralisation
    
- Opsonisation
    
- Activation du complément
    

## C. Types d’immunoglobulines

- **IgG** : majoritaires, passent placenta.
    
- **IgM** : premiers produits.
    
- **IgA** : muqueuses.
    
- **IgD** : activation LB.
    
- **IgE** : allergies / parasites.
    

## D. Phases de la réponse acquise

1. **Induction** : activation des LT/LB.
    
2. **Amplification** : prolifération clonale.
    
3. **Effectrice** :
    
    - **Réponse humorale** (anticorps)
        
    - **Réponse cellulaire** (LT8)
        

---

# VI. Réponse humorale

- LB rencontrent l’antigène → activation → différenciation en **plasmocytes**.
    
- Production d’anticorps spécifiques.
    
- Formation de **LB mémoire** → réponse secondaire rapide.
    

---

# VII. Réponse cellulaire

- **LT4** : activent macrophages, LB, LT8.
    
- **LT8** : détruisent cellules infectées par lyse (perforines/granzymes).
    

---

# VIII. Complément

Trois voies :

- **Classique** (Ac → C1q)
    
- **Lectine**
    
- **Alternative**
    

Rôles :

- Opsonisation (C3b)
    
- Inflammation (C3a, C5a)
    
- Lyse cellulaire (MAC C5b-9)
    

---

# IX. Vaccination

## A. Principe

Imitation contrôlée d’une infection → activation de l’immunité acquise → formation de cellules mémoires sans maladie.

## B. Types de vaccins

- **Inactivés** (tués)
    
- **Atténués** (affaiblis)
    
- **Sous-unités** (protéines antigéniques)
    

## C. Rappels

Renforcent la mémoire immunitaire qui s’atténue avec le temps.

---

# X. Sérothérapie

Administration directe **d’anticorps préformés**.

- **Sérop prévention** : après exposition.
    
- **Sérop curative** : infection en cours.  
    → Immunité **passive**, **immédiate**, **temporaire**.
    

---

# XI. Hypersensibilités

### **Type I (immédiate – IgE / mastocytes)**

Allergies, anaphylaxie.

### **Type II (cytotoxique IgG/IgM)**

Cytotoxicité dépendante du complément (p.ex. hémolyse auto-immune).

### **Type III (complexes immuns)**

Lupus, maladie sérique.

### **Type IV (retardée – LT médiée)**

Dermite de contact, tuberculose, sclérose en plaques.

---

# XII. Dysfonctionnements immunitaires

## A. Maladies auto-immunes

Hyperactivité immunitaire → destruction du soi.  
Ex : **sclérose en plaques** (démarche : Th17, LT CD4+, démyélinisation).

## B. Réaction allergique (type I)

1. **Premier contact** : IgE fixées aux mastocytes.
    
2. **Deuxième contact** : allergène → dégranulation → histamine → inflammation.
    

## C. Déficits immunitaires

### **Déficits innés**

- Présents dès la naissance.
    
- Peuvent toucher : immunité humorale, cellulaire, phagocytose, complément.
    

### **Déficits acquis**

- VIH (destruction LT4 → infections opportunistes)
    
- Immunosuppresseurs
    
- Chimiothérapie / radiothérapie
    
- Carences sévères
    

---

# XIII. Synthèse rapidos

- **Innée** = rapide / pas spécifique / pas de mémoire.
    
- **Acquise** = lente / spécifique / mémoire durable.
    
- **CMH I → LT8**, **CMH II → LT4**.
    
- **LB → anticorps**, **LT8 → lyse**, **LT4 → coordination**.
    
- **IgE = allergies**, **IgG = mémoire**, **IgM = premiers**, **IgA = muqueuses**.
    
- Inflammation = TNF, IL-1, IL-6 + neutrophiles + macrophages.
    
- Complément = C3b opsonisation / C5b-9 lyse.
    
- Vaccins = mémoire ; sérothérapie = anticorps immédiats mais temporaires.
