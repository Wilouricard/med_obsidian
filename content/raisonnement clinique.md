# I. Identifier le MOTIF de consultation

Toujours commencer par :

- douleur
    
- [[dyspnée]]
    
- fièvre
    
- malaise
    
- diarrhée
    
- etc.
    

C’est le **porte d’entrée** du raisonnement.

---

# II. Décrire le SYMPTÔME de manière sémiologique

Les EDN veulent que tu décrives selon les axes classiques :

- début (brutal, progressif)
    
- contexte (infection, traumatisme, effort, repos)
    
- terrain (âge, comorbidités)
    
- aggravants / soulageants
    
- signes associés
    
- notion de gravité (red flags)
    

C’est la partie **semiologie stricte**.

---

# III. Faire une HYPOTHÈSE MÉCANISTIQUE

C’est ici que les bons étudiants se distinguent :

Exemples :

- douleur thoracique → ischémique / pariétale / pleurale / digestive
    
- [[dyspnée]] → obstructive / restrictive / cardiaque / anémique
    
- fièvre → infectieux / inflammatoire / néoplasique / iatrogène
    
- douleur abdo → inflammatoire / obstructive / vasculaire / perforative
    

Cette étape est **cruciale** car elle oriente tout le raisonnement.

---

# IV. Déduire les DIAGNOSTICS DIFFÉRENTIELS pertinents

On te demande 3 à 5 diagnostics maximum, mais pertinents.

Exemples :

Pour douleur thoracique :

1. SCA
    
2. EP
    
3. Dissection aortique
    
4. Péricardite
    
5. Pneumothorax
    

Pour [[dyspnée]] :

1. [[OAP]]
    
2. BPCO décompensée
    
3. EP
    
4. Pneumopathie infectieuse
    
5. Anémie sévère
    

C’est **hiérarchisé**, concis, réaliste.

---

# V. Identifier immédiatement les SIGNES DE GRAVITÉ

C’est une étape cruciale aux EDN.  
Tu DOIS montrer que tu sais chercher la gravité avant le diagnostic.

Exemples :

- Glasgow bas
    
- [[polypnée]] sévère
    
- SpO₂ basse
    
- collapsus
    
- marbrures
    
- douleur thoracique constrictive + sueurs
    
- défense abdominale
    
- raideur méningée
    
- purpura extensif
    

C’est ce que les correcteurs veulent voir.

---

# VI. Faire le BILAN **ORIENTÉ** (jamais un catalogue)

EDN n’aime pas les listes.  
Ils veulent un bilan ciblé sur :

### Biologie

[[CRP]], hémogramme, iono, lactates, troponine, D-dimères, BNP…

### Imagerie

ECG, radio thorax, écho cardiaque, scanner TAP, écho doppler…

### Examens complémentaires

Gaz du [[sang]], BU, EFR, PL, etc.

Le tout doit être relié aux hypothèses posées.

---

# VII. INTERPRÉTER activement le bilan

Le raisonnement EDN ne s’arrête pas à “je demande X”.  
Il faut montrer :

- [[CRP]] haute + PCT basse → viral
    
- leucocytes élevés + PNN → bacterien
    
- ECG ST+ → SCA ST+
    
- [[VS]] très élevée → néoplasie / inflammatoire chronique
    
- D-dimères hauts + probabilité élevée → EP probable
    
- BU nitrites + leucocytes → IU probable
    

C’est l’étape que les étudiants ratent le plus.

---

# VIII. CONFIRMER / INFIRMER

On élimine ou on valide les hypothèses principales.

Cette étape doit être justifiée :

- “La troponine élevée confirme la souffrance myocardique”
    
- “L’hémoculture positive oriente vers…”
    
- “Le scanner TAP montre une dissection : diagnostic confirmé”
    

---

- EP grave → anticoagulation, thrombolyse si choc

# X. Argumenter la décision médicale partagée

C’est très officiel EDN :

- informer
    
- expliquer
    
- co-décider
    
- garantir autonomie
    

On ne décide jamais pour, mais **avec** le patient.

---

# XI. Réévaluer

Les EDN demandent toujours :

“Que faites-vous à H+2, H+6, H+24 ?”

La réévaluation est la **signature du bon clinicien**.
